Retale

Retale is a web and mobile application which offers a set of weekly ads based on the user location. The name “Retale” derives from wordplay between Retail and Fairy tale. The Little Red Riding Hood would run as a principal theme to communicate the brand and the Handle Basket is the signature emblem of Retale.
